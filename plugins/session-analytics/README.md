# session-analytics

セッションごとの生産性メトリクスを記録・分析するプラグイン。

## 機能

- **自動記録**: Write/Edit/Bash 操作を自動的にトラッキング
- **セッションサマリー**: 終了時にアクション数・成功率・ファイル数を集計
- **日次/週次レポート**: `/analytics` コマンドで生産性を可視化

## コマンド

| コマンド | 説明 |
|---------|------|
| `/analytics` | 今日の活動サマリー |
| `/analytics weekly` | 過去7日間の週次レポート |

## 記録メトリクス

- アクション数（Write / Edit / Bash 別）
- 成功率（コマンド失敗の追跡）
- 編集ファイル数（ユニーク）
- コマンド実行時間
- セッション数

## フック

| イベント | トリガー |
|---------|---------|
| PostToolUse(Write\|Edit\|Bash) | 操作を記録 |
| Stop | セッションサマリーを算出 |
| UserPromptSubmit | `/analytics` コマンド処理 |

## データ保存先

`~/.claude/session-analytics/`（30日超の日次データは自動削除）
