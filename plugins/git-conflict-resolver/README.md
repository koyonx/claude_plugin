# git-conflict-resolver

マージコンフリクトを検出し、両ブランチの変更コンテキストを提供するプラグイン。

## 機能

- **SessionStart**: セッション開始時に未解決のマージコンフリクトを検出
- **PostToolUse (Bash)**: `git merge/rebase/pull/cherry-pick` 後にコンフリクトを検出

## 提供情報

- コンフリクトのあるファイル一覧とコンフリクト数
- コンフリクトマーカーの内容（最初のブロック）
- 両ブランチの最近のコミット履歴

## データ保存

このプラグインはデータを保存しません（リアルタイム検出のみ）。
