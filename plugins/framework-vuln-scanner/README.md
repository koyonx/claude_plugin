# framework-vuln-scanner

フレームワーク・ランタイムのバージョンをチェックし、EOLや脆弱性リスクを警告するプラグイン。

## 機能

- **SessionStart**: セッション開始時にランタイムとフレームワークのバージョンをチェック

## チェック対象

### ランタイム
| ランタイム | 警告条件 |
|-----------|---------|
| Node.js | v16以下（EOL）、奇数バージョン（非LTS） |
| Python | 3.8以下（EOL）、Python 2 |
| Ruby | 3.0以下（EOL） |
| Go | 1.20以下（サポート終了） |

### フレームワーク
| フレームワーク | 警告条件 |
|--------------|---------|
| React | v16以下 |
| Next.js | v12以下 |
| Express | v3以下（EOL） |
| Vue.js | v2（EOL） |
| Angular | v14以下 |
| Django | v3以下 |
| Flask | v1以下 |
| Rails | v5以下（EOL）、v6（EOL間近） |

## データ保存

このプラグインはデータを保存しません（リアルタイムチェックのみ）。
