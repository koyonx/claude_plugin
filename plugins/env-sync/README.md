# env-sync

`.env` ファイルの変更を検出し、`.env.example` との同期状態をチェックするプラグイン。

## 機能

- **PostToolUse (Write|Edit)**: `.env` ファイルの変更時に `.env.example` との差分を検出

## チェック内容

1. **新規キーの検出**: `.env` に追加されたが `.env.example` に未記載のキー
2. **不足キーの検出**: `.env.example` にあるが `.env` にないキー
3. **`.env.example` 未作成の警告**: `.env` があるのに `.env.example` がない場合
4. **`.gitignore` チェック**: `.env` が `.gitignore` に含まれているか確認

## プライバシー保護

- 環境変数の**キー名のみ**を読み取ります（値は一切読みません）
- シンボリックリンクは拒否します

## データ保存

このプラグインはデータを保存しません（リアルタイムチェックのみ）。
